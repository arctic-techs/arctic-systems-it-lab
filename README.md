# Arctic Systems IT Lab

This repository documents a simulated IT environment designed to replicate real-world IT operations.

## Goals
- Build and manage a small internal network
- Deploy and configure core services (SSH, web, file sharing)
- Practice troubleshooting and incident documentation
- Develop monitoring and security tools
